Skip-Bo similates a card game of Skip-Bo that can have up to 6 players. Users have the option of playing against other humans or computers. Players can also load an existing game, or end and save a game midway. 

In order to run Skip-Bo, run the commands:
make
./SkipBoMain

If desired, users can choose to load a sample saved game by inputing the file name loadGame.txt.
